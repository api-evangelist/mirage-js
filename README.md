# Mirage JS (mirage-js)
Mirage JS is an open source API mocking library for JavaScript. It lets you build, test, and share complete frontend features without depending on a backend by running in the browser as middleware with a built-in database, factories, route handlers, and serializers. Framework-agnostic across React, Vue, Angular, Ember, and modern test runners. Mirage is itself a library, not a hosted REST API.
